# Cancer Trials Canada - Demo Script with Calendar Feature

## Introduction
"Welcome to Cancer Trials Canada - an AI-powered platform that helps patients find and match with clinical trials across Canada."

## Feature 1: AI Chat Assistant
1. **Show the chat interface**
   - "Our AI assistant helps patients find trials based on their profile"
   - Click "Best matches for me"
   - Show the trial cards with match percentages

2. **Demonstrate matching algorithm**
   - Point out the 90% match trial (green pin)
   - Click "Why Match?" to show detailed breakdown
   - Explain the 30-25-25-10-10 scoring system

## Feature 2: Interactive Map with Calendar

### Part A: Map Features
1. **Show color-coded pins**
   - Green (>75%): Excellent match
   - Orange (50-74%): Good match
   - Blue (25-49%): Fair match
   - Gray (<25%): Limited match
   - Red: User location

2. **Show routing**
   - Click any trial pin on the map
   - Real road routing appears from user location
   - Distance and match score in popup

### Part B: Calendar Feature (NEW!)
1. **Open the calendar**
   - Click the `<` button on the right side of the map
   - Calendar slides in smoothly

2. **Show statistics**
   - Point to "Active Today" count
   - Point to "Upcoming" count
   - Explain what these numbers mean

3. **Demonstrate date highlighting**
   - "Blue dates show when trials begin"
   - Scroll through March and April
   - Point out multiple highlighted dates

4. **Click today's date (March 7, 2026)**
   - Shows 2 trials starting today
   - Map automatically zooms to show these trials
   - Trial cards appear below the calendar

5. **Click future dates**
   - **March 12**: "In 5 days, this trial starts"
   - **March 21**: "In 2 weeks, this trial begins"
   - **April 7**: "Next month, this trial opens"
   - Each click zooms the map to that trial

6. **Show past trial**
   - Click January 7, 2026
   - Shows closed trial that started 2 months ago
   - Demonstrates historical tracking

## Feature 3: Trial Details
1. **Click "View Details" from map popup**
   - Shows comprehensive trial information
   - Inclusion/exclusion criteria
   - Contact information
   - Principal investigator

2. **Navigate back**
   - Click browser back button
   - **IMPORTANT**: Chat history and map state are preserved!
   - "Notice the conversation is still here"

## Feature 4: Smart Sorting
1. **Show "Best matches"**
   - Sorted by suitability score first
   - Then by distance for same scores

2. **Show "Nearest trials"**
   - Sorted by distance first
   - Then by suitability for same distances

## Feature 5: Chat Features
1. **Explain eligibility**
   - Type: "Explain eligibility for the first trial"
   - Shows detailed breakdown of why user qualifies

2. **Simplify description**
   - Type: "Simplify the first trial"
   - Shows easy-to-understand explanation

3. **Compare trials**
   - Type: "Compare trials"
   - Shows side-by-side comparison

## Feature 6: User Profile
1. **Click profile icon in header**
   - Shows current profile
   - Can update cancer type, stage, biomarkers
   - Location for distance calculations

## Closing Points

### Calendar Benefits
- **Visual Planning**: See when trials are available at a glance
- **Future Planning**: Know when upcoming trials will start recruiting
- **Historical Tracking**: See which trials have already begun
- **Map Integration**: Automatic zoom to trials on selected dates
- **Easy Navigation**: Toggle visibility to maximize map space

### Overall Platform Benefits
- **AI-Powered Matching**: Intelligent scoring algorithm
- **Real-Time Routing**: Actual road distances, not straight lines
- **Comprehensive Information**: All trial details in one place
- **Persistent State**: Navigation doesn't lose your progress
- **User-Friendly**: Intuitive interface for patients and caregivers

## Demo Flow Summary
1. Start with chat → Show AI matching
2. Open calendar → Show date features
3. Click today's date → Show active trials
4. Click future dates → Show upcoming trials
5. Click trial pin → Show routing
6. View details → Navigate back → Show persistence
7. Use chat features → Explain, simplify, compare
